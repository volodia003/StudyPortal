# StudyPortal

Educational portal sample built with WPF and SQLite.

## Quick Start
1. The database file `EduPortal.db` is created on first run.
2. (Optional) Run `Database/EduPortal.sql`.
3. Update connection string in `App.config`.
4. Install System.Data.SQLite via NuGet.
5. Build and run.

See `REPORT.md` for full details.
